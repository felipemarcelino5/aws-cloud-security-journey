# 03 - VPC Security

## Objetivo

Estudar os fundamentos de rede na AWS usando VPC, subnets, route tables, Internet Gateway, Security Groups e Network ACLs, com foco em segurança de tráfego e exposição de recursos.

## Conteúdos estudados

- AWS VPC
- CIDR
- Public Subnet
- Private Subnet
- Route Tables
- Internet Gateway
- Security Groups
- Network ACLs
- Regras inbound e outbound
- Exposição de portas

## Conceitos principais

A VPC permite criar uma rede isolada dentro da AWS. Para Cloud Security, é essencial entender como o tráfego entra, sai e circula dentro do ambiente.

O objetivo deste módulo é aprender a construir redes mais seguras, evitando exposição desnecessária de serviços para a internet.

## Laboratórios planejados

- Criar uma VPC
- Criar uma subnet pública
- Criar uma subnet privada
- Configurar route table
- Criar Internet Gateway
- Criar Security Group restritivo
- Comparar Security Group e Network ACL
- Documentar o risco de liberar acesso para 0.0.0.0/0
