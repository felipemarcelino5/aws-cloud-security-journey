# 04 - S3 Security

## Objetivo

Estudar segurança em buckets Amazon S3, com foco em bloqueio de acesso público, policies, criptografia, versionamento e boas práticas de proteção de dados.

## Conteúdos estudados

- Amazon S3
- Buckets
- Public Access Block
- Bucket Policies
- ACLs
- Encryption
- Versioning
- Object Ownership
- Data Protection
- KMS

## Conceitos principais

O Amazon S3 é usado para armazenar objetos na AWS. Um erro comum em cloud security é deixar buckets públicos ou mal configurados.

O objetivo deste módulo é entender como proteger dados armazenados no S3 e evitar exposição indevida.

## Laboratórios planejados

- Criar um bucket S3
- Bloquear acesso público
- Ativar criptografia
- Criar uma bucket policy segura
- Ativar versionamento
- Revisar permissões
- Documentar riscos de bucket público
