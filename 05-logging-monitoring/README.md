# 05 - Logging and Monitoring

## Objetivo

Estudar monitoramento e auditoria na AWS usando CloudTrail, CloudWatch e AWS Config, com foco em detecção, rastreamento de atividades e investigação de eventos.

## Conteúdos estudados

- AWS CloudTrail
- AWS CloudWatch
- AWS Config
- Logs
- Metrics
- Alarms
- Events
- API Calls
- Auditoria
- Monitoramento de atividades

## Conceitos principais

Segurança sem logs é fraca. Para investigar incidentes, é necessário saber quem fez o quê, quando, de onde e em qual recurso.

O objetivo deste módulo é aprender a coletar, consultar e interpretar eventos básicos dentro da AWS.

## Laboratórios planejados

- Ativar CloudTrail
- Gerar eventos simples na conta AWS
- Consultar histórico de eventos
- Criar métrica ou alarme no CloudWatch
- Avaliar configuração de recurso com AWS Config
- Documentar um evento suspeito ou falha de configuração
