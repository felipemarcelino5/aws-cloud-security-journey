# 06 - GuardDuty and Security Hub

## Objetivo

Estudar serviços de detecção e postura de segurança na AWS usando Amazon GuardDuty e AWS Security Hub, com foco em findings, severidade, priorização e resposta inicial.

## Conteúdos estudados

- Amazon GuardDuty
- AWS Security Hub
- Findings
- Severity
- Threat Detection
- Security Posture
- Compliance
- Prioritization
- Cloud Security Monitoring

## Conceitos principais

GuardDuty ajuda a detectar atividades suspeitas na conta AWS. Security Hub centraliza findings e padrões de segurança.

O objetivo deste módulo é aprender a interpretar alertas de segurança, priorizar riscos e documentar ações de resposta.

## Laboratórios planejados

- Ativar GuardDuty
- Analisar findings de exemplo
- Classificar severidade
- Ativar Security Hub
- Revisar recomendações de segurança
- Criar relatório simples de findings
- Documentar ação corretiva para um alerta
