# AWS Cloud Security Journey

Este repositório documenta minha jornada de estudos para me tornar um profissional Junior AWS Cloud Security, com foco em IAM, VPC, S3 Security, CloudTrail, GuardDuty, Security Hub, resposta a incidentes e espanhol técnico.

## Objetivo

Construir uma base prática em Cloud Security usando AWS como cloud principal, criando projetos documentados que possam ser apresentados em processos seletivos para vagas na Espanha.
